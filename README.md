Color schema/theme: blue - #0b23d5, White - #ffffff

Typography: Playfair Display, roboto


The Handcrafted Haven website will use a clean and simple layout designed to highlight handmade products. The main pages will include a header with navigation, a product browsing section, and a footer with additional links. The home page will feature a hero section introducing the platform, followed by categories, featured products, and artisan highlights. Product pages will display images, descriptions, prices, seller information, and reviews.Cards will be used to display products and seller profiles, creating an organized and visually appealing layout that is easy to navigate on both desktop and mobile devices.
